# Dataset

The project uses brain microscopy images for evaluating deep learning-based brain cell detection and segmentation models.

The complete dataset has not been uploaded due to storage limitations and licensing considerations.

This folder contains only a few representative sample images used for demonstration.

Image Types
- Brain microscopy images
- High-resolution cell images
- Cellular structures
- Sample input images for testing

Purpose
These images are used for demonstrating:
- Cell Detection
- Cell Segmentation
- Model Comparison

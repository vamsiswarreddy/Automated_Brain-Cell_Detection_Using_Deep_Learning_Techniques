# Google Colab Notebook

This folder contains the Google Colab notebook used for implementing the proposed brain cell detection framework.

## Notebook Features

- Dataset loading and preprocessing
- Brain microscopy image visualization
- YOLOv8-based brain cell detection
- BDA contour-based detection
- Comparative analysis of detection models
- Output visualization
- Performance evaluation

The notebook can be executed directly in Google Colab without requiring local GPU setup, provided the required dependencies and dataset are available.
